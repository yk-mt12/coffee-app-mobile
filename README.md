# coffee-app-mobile
## 概要
珈琲のためのモバイルアプリケーション

## 使い方
1. 珈琲ドリップをする際に使います。
2. 湯量 or 豆量を入力すると、ratioにある比率を元に適した計算結果を表示します。
3. 右上の歯車マークを押すと、モーダルが表示され比率を変更することが可能です。
4. 下のタブを選択することで、アイスコーヒーとカフェオレを切り替えることが可能です。

## QRコード
このQRコードをiPhoneなどで読み取ると、Expo goというreact nativeの実行環境に飛びます。
